## 🎯 About the Lab

This lab introduces students to modern interactive technologies including:

* 🎮 Unity & Interactive 3D Applications
* 🥽 Virtual Reality (VR)
* 📷 Computer Vision
* 👁️ Eye-tracking Interaction
* 📱 Augmented Reality (AR)

Each workshop includes:

* Full Unity project
* Scripts (reusable where applicable)
* Assets & supporting material

## ▶️ How to Use

### Option 1 — Open Full Project

1. Open **Unity Hub**
2. Click **Open Project**
3. Select:

   ```
   workshop-X/Unity Project - ...
   ```
4. Open the main scene and press Play

---

### Option 2 — Use Scripts Only

* Navigate to:

  ```
  workshop-X/Scripts Only ...
  ```
* Import scripts into your own Unity project
